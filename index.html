Burger City

Мобильное приложение для сети бургерных. Конструктор бургеров. Онлайн заказ с адресной доставкой.

Стек технологий: Python   Django   iOS   Android  

Pizzer

Приложение пицерии. Возможность выбрать и забронировать столик, сделать заказ и оплатить через мобильное приложение.

Стек технологий: Python   Django   Django Rest Framework   Postgresql   Redis   iOS   Android   Celery  

Sorter Shop

Web-приложение для организации своих заметок. Позволяет создавать карточки, добавлять в них чек-листы, перемещать по папкам, делиться с другими пользователями, настраивать нотификации.

Стек технологий: Python   Django   Django Rest Framework   Redis   MySQL   AgularJS   HTML   CSS   SASS   Bootstrap   Celery
