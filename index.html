Burger City

Pizzer

Приложение пицерии. Возможность выбрать и забронировать столик, сделать заказ и оплатить через мобильное приложение.

Стек технологий: Python   Django   Django Rest Framework   Postgresql   Redis   iOS   Android   Celery  

Sorter Shop

Web-приложение для организации своих заметок. Позволяет создавать карточки, добавлять в них чек-листы, перемещать по папкам, делиться с другими пользователями, настраивать нотификации.

Стек технологий: Python   Django   Django Rest Framework   Redis   MySQL   AgularJS   HTML   CSS   SASS   Bootstrap   Celery  

TicketToDo

Web-приложение для организации своих заметок. Позволяет создавать карточки, добавлять в них чек-листы, перемещать по папкам, делиться с другими пользователями, настраивать нотификации.

Стек технологий: Python   Django Rest Framework   JavaScript   Postgresql   Redis   AgularJS   Bootstrap   Celery  

City Taxi App

Приложение для сервиса такси. Клиентская часть позволяет вызывать такси по адресу, выбирать уровень автомобиля, оплачивать поездку онлайн. Водительская часть позволяет брать заказы, видеть их на карте, выводить заработанные деньги на карту.

Стек технологий: Python   Django   Django Rest Framework   Postgresql   Celery   PostGis   Memcached  

Онлайн магазин одежды

Приложение для магазина одежды. Покупка товаров онлайн, история заказов, уведомление об ассортименте.

Стек технологий: Python   Django   Django Rest Framework   Redis   MySQL   Celery  

SportPit - магазин спортивного питания

Магазин спортивного питания. В админке реализована подробная статистика продаж, учет ассортимента, поставки, клиентская база, механизм рассылок.

Стек технологий: Python   Django   Redis   MySQL   Bootstrap   Celery  

Kvartirkov

Сервис подбора объявлений посуточной аренды жилья в Украине. Разрабатывался основной функционал, кабинет риелтора, поиск и фильтрация объявлений, биллинг.
